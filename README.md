# Game Library

> Мобильное приложение для управления библиотекой игр в стиле Steam

---

## 📋 Описание

**Game Library** — это простое мобильное приложение. Приложение позволяет просматривать каталог игр, изучать подробную информацию о них и отслеживать прогресс выполнения игровых заданий.

### 🎯 Тематика

Приложение посвящено **видеоиграм** и вдохновлено дизайном популярной платформы **Steam**:
- 🎨 Тёмная цветовая схема с синими акцентами
- 🕹️ Карточки игр с обложками, рейтингом и жанрами
- 📊 Система прогресса и достижений в профиле
- 🔍 Удобный поиск по каталогу (просто заглушки)

---

## ✨ Возможности

### 🏠 Главная страница
- Список популярных игр в виде карточек
- Быстрый поиск по названию (просто заглушки)
- Плавные переходы к деталям игры

### 📚 Библиотека
- Сеточное отображение всех игр (2 колонки)
- Адаптивные карточки с обложками

### 🎮 Детали игры
- Крупное изображение обложки
- Название, жанр и рейтинг
- Подробное описание
- Кнопка "Запустить" (просто заглушки)

### 👤 Профиль пользователя
- Аватар и статистика игрока
- Система заданий (5 галочек)
- Прогресс-бар с анимацией заполнения
- Награда при выполнении всех заданий 🎁

### 🧭 Навигация
- Нижняя панель навигации (Bottom Bar)
- Плавные переходы между экранами
- Адаптивное скрытие панели на экранах деталей
- Поддержка навигации назад
